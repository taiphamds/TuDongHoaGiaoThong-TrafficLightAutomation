# Hệ thống giao thông thông minh – Dự án cá nhân
Dự án đề suất thuật toán tính số giây đèn xanh qua số lượng phương tiện chờ đèn đỏ tại riêng mỗi tuyến đường tại ngã tư(đọc '(Summary)GiaoThongThongMinh.pdf' để biết rõ hơn)
## Các tính năng chính

- Phát hiện phương tiện bằng YOLOv10
- Thuật toán động để điều chỉnh thời gian đèn xanh
- Giao diện người dùng thân thiện với người dùng có mô phỏng giao thông trực tiếp

## Thư viện

- Python
- YOLOv10 (Tầm nhìn máy tính)
- OpenCV
- Tkinter (GUI)
- NumPy
